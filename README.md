## How to run

1. Clone this repository
2. Install libraries:  
pip install -r requirements.txt
3. Download the dataset from Kaggle:  
[https://www.kaggle.com/datasets/csafrit2/maternal-health-risk-data](https://www.kaggle.com/datasets/csafrit2/maternal-health-risk-data)
4. Rename it to: maternal_health_rish.csv
5. Place it in the project folder
6. Run the app:  
python -m streamlit run [app.py](http://app.py)

## Project Structure

- [app.py](http://app.py) - main Streamlit app
- bmi_module.py - BMI calculator (Yada)
- edd_module.py - delivery date calculator (Yada)
- risk_module.py - risk prediction AI model (Team)
